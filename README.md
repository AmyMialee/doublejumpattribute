# Double Jump Attribute
Double Jump Attribute adds a new attribute giving the players extra midair jumps.  
This attribute can be applied like any other, to a cap of 1024.  

<img src="https://github.com/AmyMialeeMods/double-jump-attribute/raw/main/assets/jump.png" alt="Double Jump.">  

To add Double Jump Attribute to your project as a dependency you need to add the following to your `build.gradle`:
```gradle
repositories {
	maven { url = "https://maven.willbl.dev/releases" }
}

dependencies {
	modImplementation "amymialee:doublejumpattribute:1.0.8+1.19"
}
```
